# AutoSafeX - TestBench Generator

이 프로젝트는 최상위 하드웨어 모듈(Verilog 등) 코드를 읽어와 정형화된 **테스트벤치(TestBench) 파일의 뼈대를 자동으로 작성**해 주는 프로그램의 루트 저장소입니다. 

사용자가 직접 코딩해야 했던 클럭 생성 구문, 리셋 시퀀스 동작, DUT 인스턴스화, 포트 연결, 랜덤 입력값 할당 등의 반복 작업을 GUI 설정을 통해 즉각적으로 자동화하는 것을 목표로 합니다.

## 개발 목적 및 협업 가이드
이 저장소는 자동 테스트벤치 생성기를 팀 혹은 여러 명이 협력하여 함께 발전시키기 위해 만들어졌습니다.
- **동시 수정과 버전 관리:** GitHub(혹은 로컬 Git 시스템)를 활용하여 소스 코드를 통합/수정하기 쉽도록 프로젝트 디렉토리 폴더(src, docs, tests, templates)구조를 잡아 두었습니다.
- 프로젝트의 전체적인 기능 정의는 `docs/specification.md` 문서를 먼저 읽어보시길 권장합니다.

## 프로젝트 디렉토리 구조
- `docs/`: 프로그램 사양서(`specification.md`) 등 기획 및 구조 문서 보관
- `src/`: 프로그램의 실제 파이썬/스크립트 소스코드 (추후 Verilog 파서, 클럭/리셋/랜덤 생성 로직, GUI 프론트엔드가 추가될 위치)
- `templates/`: 결과 생성을 위한 뼈대가 되는 베이스 템플릿 코드. (여기 지정된 텍스트 뼈대를 읽어와 변수만 치환하여 생성)
- `tests/`: 프로그램 자체의 로직들을 검증하기 위한 단위 테스트 스크립트 모음

## 시작하기
이 저장소를 클론(`git clone`)하여 본인의 기능을 추가하고 커밋 단위로 작업 내역을 공유할 수 있습니다. 
만약 새로운 코드를 짤 뼈대나 템플릿이 필요하다면 `templates` 폴더 내에 규칙을 정해 텍스트 파일을 만들면 됩니다.
